# Fundamentals of Artificial Intelligence (BI-ZUM)

Welcome to my Artificial Intelligence portfolio! This repository contains implementations of classic AI algorithms and problem-solving techniques developed for the **BI-ZUM** course. The focus is on algorithmic thinking, state-space exploration, and automated decision-making.

## 🚀 Repository Overview

The projects in this repository explore different paradigms of artificial intelligence, moving from exact, systematic algorithms to probabilistic optimization and automated planning.

### [Task 01: Systematic State-Space Search](./01/)
A deep dive into exact algorithms guaranteed to find optimal paths or solutions.
* **Key Focus:** Implementing blind and heuristic search algorithms (e.g., BFS, DFS, Greedy-Search, Random-Search, A*) to navigate complex state spaces efficiently.

### [Task 02: Non-Systematic (Local) Search](./02/)
Tackling NP-hard problems where systematic search is too slow or computationally impossible.
* **Key Focus:** Utilizing stochastic and local optimization algorithms (e.g., Hill Climbing, Simulated Annealing, Genetic Algorithms) to find "good enough" solutions in massive search spaces.

### [Task 03: Automated Planning](./03/) 🚧 *(Work in Progress)*
Moving beyond static graphs to dynamic environment solving.
* **Key Focus:** Formulating problems using actions, preconditions, and effects to allow an AI agent to autonomously generate a sequence of steps to reach a goal state.
