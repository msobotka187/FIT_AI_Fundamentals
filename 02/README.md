### Úkol:
  - Zvolil jsem **Symetrického obchodního cestujícího**
  - Pomocí **Simulated Annealing** (Simulovaného žíhání)

### Technologie:
  - **Jazyk:** C++
  - **Vizualizace:** Raylib

### Implementace:
  - **Backend:** v C++ vytvoříme třídu pro město
  - **Žíhání:** implementujeme algoritmus a otestujeme v konzoli
  - **Propojení s vizualizacemi:** Hotový algoritmus překreslíme v okně
  - **Animace:** Nějak to vymyslíme, aby to pěkně vypadalo, jak to probíhá ten algo, ještě vymyslim.
